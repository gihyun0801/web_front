input 태그

웹 문서에서 사용자가 입력할 수 있는 양식을 제공하는 태그

text 관련된 input 태그

type="text"

      - 한 줄 짜리 문자열(텍스트)를 입력할 수 있는 입력 상자
      - input 태그의 type의 기본값으로, 생략가능
      - inline 형식 (모든 input 태그가 inline)
  
    
text 관련 input 태그의 공용 속성

    - size : 입력상자 크기
    - maxlength : 입력 받는 텍스트의 최대길이
    - placeholder : 입력상자안에 텍스트를 미리 보여주는 태그

form 태그

  - div 와 같은 영역(block 형식)
  - 내부에 작성된 input 태그의 값을 서버로 전달하는 역할

  [속성]

  1) action : 내부에 작성된 값을 전달할 서버 또는 페이지 주소를 작성
  2) name : form 태그의 이름을 지정하여 form 태그 구분 (JAVASCRIPT 사용)
  3) method(방식) : 서버로 데이터 전달하는 방식을 지정하는 속성
  4) target : action에 지정된 주소를 현재 탭에서 열거냐 새 탭 에서 열거냐 지정

select 태그, option 태그

  - 기본 값으로 처음부터 option안에 있는 내용이 선택 되어지게 하려면
    option 태그 에 selected 를 주면된다.


  - select 제출 시
    option 태그에 value 가 있으면 : value 속성안에 작성된 값이 넘어가고
    option 태그에 value  가 없으면 : option 태그 내부에 작성된 글이 넘어간다

  -checked 속성

  input checkbox , radio 에 주는 속성으로 checked 를 주면 기본으로 사이트를 새로고침 하였을때 체크 되어 있는 속성이다
