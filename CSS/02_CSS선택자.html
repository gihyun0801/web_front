기본 속성 선택자

    태그에 작성된 특정 속성을 선택하는 선택자.

    작성법
    선택자[속석명="속성값"]{css 코드 :}

    (참고)
    1) 선택자는 생략할 수 있다.

    2) "속성값" 양쪽의 쌍다옴표("")는
    홀따옴표('')로 변경할 수 있다.

반응 선택자

    사용자의 움직임에 따라 반응하여 스타일이 달라지는 선택자
    (마우스 오버, 클릭 유지..)

    - 요소를 클릭하고 있는 경우(:active)
    - 요소에 마우스가 올라가는 경우(:hover)

상태 선택자

    입력 양식(input)의 상태에 따라 선택되는 선택자.

    :focus -> 요소에 초점이 맞춰졌을 때
    :checked -> 요소가 체크 되었을 때

    :enabled / :disabled
    -> 요소가 사용 가능한/불가능한 상태

동위 선택자

    동위 관계(동등한 위치 == 형제 관계)에서
    뒤(다음)에 위치한 요소를 선택하는 선택자.

    1) A선택자 + B선택자 {css 코드 ;}
    A 바로 뒤(다음)에 있는 B요소 하나를 선택한다는 것이다

    2) A선택자 ~ B선택자 {Css 코드 ;}
    A 뒤에 있는 모든 B요소를 선택 (~틸드(Tilde))
